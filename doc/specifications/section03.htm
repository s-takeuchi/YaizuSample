3. Deployment

3.1 AWS deployment

Deployment procedures for AWS are described below.
  1. Install Serval server
  2. Allow port number 8081 for EC2 security group.
  3. svrUrl change pf stkcommon.js (for cross domain access)
  4. sample.conf servicehost=IP address
  5. Restart Serval server
  6. Update Serval client
  7. stkcommon.js ; change svrhost="https://commander.red-squirrels.net"
